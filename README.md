# Siphon
YT-DLP wrapper with playlist watcher, scheduler and auto renaming functions built for music.

Features/Goals:
- Download entire playlists or single videos.
- Watch for changes in playlist and auto download newly added videos/music.
- Download on a schedule automatically.
- Rename downloads using metadata or external API (MusicBrainz).
- Containerized (Built for Unraid).
- Maybe add video download support or the option to select download format later.
